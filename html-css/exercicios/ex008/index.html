Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag B.

Nesta frase, tenho um termo em destaque usando a tag STRONG.

Itálico / Ênfase

Nesta frase, temos um termo em ênfase, usando a tag EM.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Porém, a tag <style> dentro da tag <mark> só irá alterar o trecho específico. Para alterar todos os "mark's", devemos adicionar o comando em <head>

Texto grade e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado, com a tag U.

Texto sobrescrito

Para inserir coisas do tipo: x20+3

Texto subscrito

Para inserir coisas do tipo: H2O
